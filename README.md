## add full icu to node for i18n
```sh
npm install -g full-icu
```

## Setup
```sh
yarn install
```

## run
It's served in http://dev.btcchina.com:3010 .
```sh
yarn start
```

## test
```sh
yarn test
```

## build
It's served in ./static folder.
```sh
yarn run build
```
```sh
yarn run build:staging
```
```sh
yarn run build:dev
```

## client rendering
```sh
yarn run client
```

## server rendering
```sh
yarn run server
```
